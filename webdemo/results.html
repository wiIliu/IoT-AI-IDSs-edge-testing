Interpretation Tips

  • Macro metrics average classes equally; helpful under class imbalance.
  • ROC curves compare TPR vs FPR; higher AUC indicates better separability.
  • PR curves are often more informative with heavy class imbalance.
  • The confusion matrix highlights common confusions to guide error analysis.
