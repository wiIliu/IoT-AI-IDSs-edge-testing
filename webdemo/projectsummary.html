Project Summary

A high-level overview of our IDS in IoT capstone project, comparing traditional machine learning models and deep learning architectures for detecting malicious IoT network traffic.

Goals of the Project

  • Explore two AI approaches for detecting malicious IoT network traffic:
    • Machine Learning (ML) models
    • Deep Learning (DL) models
  • Build and evaluate both families of models separately to directly compare:
    • Accuracy
    • Performance
    • Real-world practicality in cybersecurity settings
  • Analyze how ML and DL models handle the same IoT dataset (RT-IoT-2022).
  • Identify each approach’s strengths and weaknesses for intrusion detection, highlighting trade-offs between interpretability, complexity, and deployment cost.
