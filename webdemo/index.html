Intrusion Detection in RT-IoT-2022

A practical comparison of traditional ML (ensembles) vs DL (CNN) for binary and multi-class classification.

What is this?

This site summarizes our pipeline and showcases comparative results. Training occurs offline (Python / PyTorch / scikit-learn). The exported data/results.json is loaded here to render tables and charts.

Use the Results page to interact with metrics (accuracy, macro P/R/F1), confusion matrices, ROC and precision–recall curves for each model.

How to update results

  1. Run your training notebooks/scripts.
  2. Export metrics to webdemo/data/results.json.
  3. Start the server: node server.js.
  4. Refresh /results.html.
