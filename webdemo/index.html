Project Overview

Our project focused on building artificial intelligence models to classify Internet of Things (IoT) network data as either benign or malicious. We implemented and compared two approaches: traditional machine learning (ML) methods using models such as XGBoost and decision trees, and a deep learning (DL) method using an RNN-LSTM model that captures sequential dependencies. By evaluating both approaches, we identified which model performed better as an intrusion detection system (IDS) in terms of accuracy, precision, recall, and overall robustness.

    Overview

  • Binary classification: benign vs. malicious traffic
  • Metrics: Accuracy, Precision, Recall, F1, ROC‑AUC
  • Deliverables: trained models, evaluation, and a demo-ready site
