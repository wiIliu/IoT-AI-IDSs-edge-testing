Intrusion Detection in RT-IoT-2022

A comparative evaluation of traditional machine learning (ML) methods and deep learning (DL) architectures for binary and multi-class intrusion detection.

What is this project?

This project examines how different intrusion detection approaches perform when applied to the RT-IoT-2022 dataset — a modern dataset of labeled IoT network traffic containing both benign activity and multiple cyberattack types. The goal is to evaluate whether lightweight ensemble-based ML models or deep learning architectures provide stronger detection performance under binary and multi-class conditions.

The system includes both traditional ML and deep learning approaches:

  • Machine Learning Models: Random Forest, XGBoost, and CatBoost
  • Deep Learning Models: CNN classifiers (binary and multiclass)
  • Enhanced Architectures: CNNs incorporating self-attention mechanisms

All models were trained offline and evaluated using metrics such as accuracy, precision, recall, F1-score, MCC, ROC-AUC, and confusion matrix visualizations to compare performance across multiple attack classes.

To explore the findings, navigate to the Results CNN and Results ML pages.
