Results for Deep Learning Models

The deep learning models implemented in this project were trained, validated, and evaluated using the RT-IoT-2022 dataset. These models were designed to perform both binary intrusion detection (benign vs. malicious traffic) and multiclass classification across multiple attack types. The evaluation below highlights the performance of the deployed convolutional neural network (CNN) architectures.

Multiple deep learning configurations were tested, including standard 1D and 2D CNN architectures as well as enhanced versions incorporating attention mechanisms to improve feature learning and classification accuracy. Key evaluation metrics such as accuracy, precision, recall, and F1-score were computed to assess detection performance in both binary and multiclass scenarios.
