Technologies Used

Core tools, languages, and frameworks used to build and evaluate our intrusion detection system for IoT network traffic.

  • GitHub – repository management and collaboration.
  • VS Code (IDE) – primary development environment for code and experiments.
  • Python (TensorFlow / PyTorch) – language and deep learning frameworks used to build convolutional neural network (CNN) models and traditional machine learning baselines.
  • HTML / CSS / JavaScript – used to design and implement the project website and interactive results interface.
  • Ensemble models – traditional machine learning approaches such as Random Forest, XGBoost, and CatBoost.
  • Convolutional Neural Networks (CNNs) – deep learning architectures for both binary and multi-class intrusion detection.
